# Discord tailwind

Project to master tailwind. To see the result, please visit https://franciscojaviermartin.github.io/Discord-tailwind/

## Project Setup

```sh
pnpm install
```

### Compile and Hot-Reload for Development

```sh
pnpm dev
```

### Type-Check, Compile and Minify for Production

```sh
pnpm build
```

### Lint with [ESLint](https://eslint.org/)

```sh
pnpm lint
```
